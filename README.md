Проект использует исключительно компьютерное зрение (анализ изображения с экрана). Мы не читаем память игры, не модифицируем игровые файлы и не предоставляем возможностей wallhack, aimbot или других функций, нарушающих целостность игры. Это исследовательский проект по созданию автономного ИИ-агента.
